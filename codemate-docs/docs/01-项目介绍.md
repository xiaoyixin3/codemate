# CodeMate 技术社区后端系统

## 一、项目简介

CodeMate 是一个面向程序员的技术社区后端系统，基于 Spring Boot 构建，支持用户登录注册、文章发布、评论回复、点赞收藏、用户关注、消息通知、内容搜索和排行榜等核心功能。

本项目以技术内容社区为业务场景，重点实践 Java 后端开发中的用户认证、接口设计、数据库建模、缓存优化、消息通知、内容检索和排行榜统计等能力。

## 二、项目定位

项目类型：Java 后端项目 / 技术社区系统 / 内容管理系统

适用场景：

- 程序员技术文章分享
- 用户评论互动
- 文章点赞收藏
- 用户关注关系
- 热门文章排行榜
- 站内消息通知

## 三、技术栈

后端技术：

- Spring Boot
- MyBatis / MyBatis-Plus
- MySQL
- Redis
- JWT
- Maven
- Liquibase
- AOP
- 定时任务
- 拦截器

## 四、核心功能

- 用户注册与登录
- JWT 登录态校验
- 文章发布、编辑、删除、查询
- 文章分类与标签管理
- 评论与回复
- 点赞、收藏、关注
- 站内消息通知
- 热门文章排行榜
- 热点文章缓存
- 数据库版本初始化管理

## 五、项目亮点

1. 基于 JWT + 拦截器实现用户登录态校验与接口权限控制。
2. 使用 Redis 缓存热点文章、用户信息和排行榜数据，降低 MySQL 查询压力。
3. 基于 Redis ZSet 实现文章热度排行榜，提高排行榜查询性能。
4. 设计站内消息通知模块，在评论、点赞、关注等行为发生后生成通知记录。
5. 使用 Liquibase 管理数据库初始化脚本，保证本地环境快速启动。
